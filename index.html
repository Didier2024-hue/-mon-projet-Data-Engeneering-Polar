POLAR ACCESSLINK API

Secure OAuth2 connection to retrieve training sessions, heart rate, distance, and biometric data directly from the Polar ecosystem.

Connexion OAuth2 sécurisée pour récupérer les séances d'entraînement, fréquence cardiaque, distance et données biométriques depuis l'écosystème Polar.

PYTHON ETL PIPELINE

Custom ETL scripts handle data extraction, transformation and loading into InfluxDB with proper timestamp management and field mapping.

Scripts ETL personnalisés pour l'extraction, la transformation et le chargement dans InfluxDB avec gestion des timestamps et mapping des champs.

INFLUXDB — TIME-SERIES

Purpose-built time-series database optimised for sports metrics. Flux queries enable complex aggregations across training history.

Base de données temporelle optimisée pour les métriques sportives. Les requêtes Flux permettent des agrégations complexes sur l'historique d'entraînement.

CUSTOM MCP SERVER

A bespoke Model Context Protocol server exposes InfluxDB tools to Claude AI, enabling real-time dynamic data access — not static ML inference.

Un serveur MCP (Model Context Protocol) sur mesure expose les outils InfluxDB à Claude AI, permettant un accès dynamique aux données en temps réel.

CLAUDE AI — ANTHROPIC

Claude dynamically selects and calls the right MCP tools based on user questions, then synthesizes training insights from live data.

Claude sélectionne et appelle dynamiquement les bons outils MCP selon les questions, puis synthétise des insights d'entraînement à partir des données en direct.

FLASK + SYSTEMD

Flask web backend serves the coaching interface. Deployed as a systemd service on Ubuntu for reliable auto-start and production stability.

Backend web Flask pour l'interface coaching. Déployé comme service systemd sur Ubuntu pour un démarrage automatique et une stabilité en production.

Polar Coach AI is a personal project designed to showcase end-to-end Data Engineering skills — from API integration and ETL pipelines to time-series storage, visualisation, and AI-powered interfaces.

Polar Coach AI est un projet personnel conçu pour démontrer des compétences Data Engineering de bout en bout — de l'intégration API et des pipelines ETL au stockage temporel, à la visualisation et aux interfaces propulsées par l'IA.

The project runs on a Linux Ubuntu VM with full production setup: UFW firewall, systemd service, and a custom MCP server bridging live InfluxDB data with Claude AI.

Le projet tourne sur une VM Linux Ubuntu avec une configuration production complète : pare-feu UFW, service systemd, et un serveur MCP personnalisé reliant les données InfluxDB en direct à Claude AI.
